# SIMATRIX Official Site

Static official website for SIMATRIX / Symbiotic Matrix.

## Structure

- `index.html` - home page
- `capabilities.html` - capabilities
- `work.html` - practices
- `about.html` - about and contact
- `methodology.html` - redirect compatibility page
- `styles.css` - site styles
- `script.js` - navigation and multilingual switching
- `assets/` - static assets

The site is pure static HTML/CSS/JS and does not require a build step.
